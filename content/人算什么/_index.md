# 人算什么 

史百克弟兄《What is Man》(人算什么)，1939年第一版 (紧接着倪弟兄访欧讲《正常的基督徒生活》并与他交通)，论及成圣的实质与盲区，是不可错过的上乘佳作，建议读英文版:

http://www.austin-sparks.net/english/books/what_is_man.html (1951年修订版)

《人算什么》这本书，没有争论灵魂体是否三元论，却从圣经与生命的经历来说明灵的真实存在与意义。我们对灵与魂的认识混淆，就会造成重生与成圣错误了解，阻碍我们灵命长进。灵与魂有区别，我们借着灵与神有五重的关系。但愿我们认识自己的本相，以简单、谦卑、无己的心跟随主。成圣的路在于信而顺服，倚靠神，好叫灵与圣灵联合。因此，我们必须更新的认识十字架，去抵挡那迷惑、欺骗人的仇敌对魂所作假冒的事。这样，我们为复兴的祷告和为灵命进深的努力，就能够落实。

# 前 言

人是否由三部分组成本为极具争议性的论题，我要对读者们说，本书并非要参与这争论。辩论恰恰与本书的重要主张相抵触，因为属灵的事物只能藉启示（而绝非藉辩论）进入。我的确不盼望任何没有在现实及属灵事物真实操练过的人来读这本书。我请求读者有个敞开的心，好使真理的灵或许有可能使用书中所写的来光照启迪。

我并不宣称这是什么专家的见识，也非研究这题目本身的成果，而是多年来经过多面观察及在基督徒的经验中所得的结论。

本书是由于保罗在二千年前的一个祷告而开始------「求我们主耶稣基督的神，荣耀的父，将那赐人智慧和启示的灵赏给你们，使你们真知道祂，并且照明你们心中的眼睛。」 (弗一17-18)

史百克，生于英国伦敦。17岁在苏格兰格拉斯哥（Glasgow）街头听见福音蒙恩。1923至1926年间，他曾是宾路易师母的年轻同工。1926年，在伦敦东南贵橡路创办了\[贵橡基督徒交通中心\]，专一地发表主托付他的职事，并主办了一份很有属灵供应的基督徒刊物《见证人与见证》。

史百克被后人称为内里生命派最杰出的属灵导师。他的职事是\[启示基督自己，叫人认识神永远的目的是叫万有服于元首基督之下；万有是为着基督、出乎基督、并藉着基督，基督是一切的中心\]。

史百克一生以讲道和文字服侍众圣徒，影响力遍及世界各地，得其帮助者其数众多，包括印度的巴新和中国的倪柝声。当初倪柝声弟兄在离英回国的途中，写一封信给史百克，说到他回到中国之后，将会感到非常孤单，因为似乎没有一个人能像史百克这样地帮助他。

纵观当今末世时日，基督教世界能发表有属灵分量和深度的文字信息已不多见，所以越发显出史百克弟兄信息的宝贵，愿主藉着这一极有属灵价值的文集，开启今日众圣徒的眼睛，能更深明白神永远的旨意和祂终极的计划之所在。愿一切赞颂都归给祂，阿们！
